# Colors

## Rules

- Use semantic role, not raw color value
- **Brand** — primary actions, links, focus rings
- **Neutrals** — text, backgrounds, borders, dividers
- **Accents** — status feedback: success, warning, error
- **Alpha** — text opacity variants (88%, 65%, 45%, 25%)
- **Additional** — app-level semantic aliases and AG Grid mappings
