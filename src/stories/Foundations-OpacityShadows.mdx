# Opacity & Shadow

Токены прозрачности и глубины.

## Правила

- Opacity использовать только для системных состояний
- Shadow применять только для layer separation
- Не создавать кастомные тени без нового уровня в системе

## Opacity

<div style={{ display: "grid", gap: 12, marginBottom: 32 }}>
  {opacity.map((token) => (
    <div
      key={token.name}
      style={{
        display: "grid",
        gridTemplateColumns: "180px 120px 80px",
        gap: 16,
        alignItems: "center",
      }}
    >
      <div>{token.name}</div>
      <div
        style={{
          width: 80,
          height: 40,
          background: `rgba(68, 140, 192, ${parseInt(token.value, 10) / 100})`,
          borderRadius: 8,
          border: "1px solid rgba(0,0,0,0.06)",
        }}
      />
      <div>{token.value}</div>
    </div>
  ))}
</div>
