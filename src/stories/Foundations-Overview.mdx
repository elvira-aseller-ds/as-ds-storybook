# Foundations

Базовый слой дизайн-системы.

Содержит системные токены, которые используются во всех:
- компонентах
- паттернах
- фичах

## Состав

- Colors
- Spacing & Radius
- Size
- Typography
- Opacity & Shadow
- Other

## Правила

- Использовать только системные значения
- Не создавать новые токены без необходимости
- Один токен = одна роль в интерфейсе
- Сначала выбирать существующий semantic token
