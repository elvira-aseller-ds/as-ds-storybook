# Size

Размерные токены для элементов и контейнеров.

## Использование

- controls
- icons
- cells
- cards
- layout blocks

## Правила

- Размер определяется token scale
- Одинаковые сущности должны иметь одинаковый размер
- Не использовать произвольные width / height без причины
