# Aseller Design System

Единая система интерфейсов для платформы Amazon sellers.

## Принцип

**Экран = Паттерн + Вариант**

## Архитектура

- Foundations
- Patterns
- Components
- Features
- Assets

## Как работать

1. Найти нужный паттерн
2. Проверить, есть ли нужный вариант
3. Использовать существующий компонент
4. Не создавать новый экран, если это вариант существующего паттерна
